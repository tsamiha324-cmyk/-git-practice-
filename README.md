# -git-practice-
Objective:
The objective of this lab is to create a simple C++ program that stores student information and shows it on the screen. In this program, string variables are used to store the student's name, ID, section, department, and university. This lab also helps us practice using variables, the cout statement, and the basic structure of a C++ program.

Outcome:
The program ran successfully and showed all the student information in a clear and organized format. It displayed the student's name, ID, section, department, university, and a welcome message. Through this lab, I learned how to use string variables, print output using cout, and write a simple C++ program without any errors.
